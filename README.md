# Sukkergris – Web Application

This project was developed as a final exam for the course Application Development 1 
in the bachelor program Multimedia Design and Technology.

The project explores how a playful visual identity can be combined with functional
e-commerce logic and user interaction.

## Features
- Multiple-page web application
- User registration and login
- Conditional content based on login status
- Product filtering for non-logged-in users
- User ratings and comments
- Profile picture continuity across the application

## My role
I was responsible for:
- Homepage design and development
- User registration and login
- Profile picture continuity
- Restricting access to certain products for non-logged-in users
